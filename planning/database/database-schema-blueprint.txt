INDEXES
--------------------------------------------------------------------------------
1. Index on all foreign keys (e.g., idx_employees_dept_id, idx_employees_manager_id)
2. Index on date columns for range queries (attendance.log_date, salary_slips.month_year)
3. Index on status columns (leave_requests.status, candidates.status)
4. Composite unique indexes where specified in constraints
5. Full‑text search indexes on candidate names, document titles (if using PostgreSQL)
6. GIN indexes on JSONB columns (metadata, old_data, new_data) for advanced queries
7. Index on email, emp_code, user_id, etc. for quick lookups
